# Commands

`ban [member] [reason]`

> Ban a member 

`ban sudo <user_id> [reason]`

> Ban a user that is not in the actual guild 

`work `

> Work and earn money 

`unban <user_id>`

> Unban a user 

`leaderboard `

> See users lb 

`kick [member] [reason]`

> Kick a member 

`mute [member] [reason]`

> Mute a member 

`gun `

> Make a gun battle 

`unmute [member]`

> Unmute a member 

`lock [channel]`

> Lock a Text Channel 

`unlock [channel]`

> Unlock a Text Channel 

`hungergames [members]`

> Create an Hunger Game! If members is none, bot will choose 8 random members. Use `,` to separate players, Ex. `hungergames lmao, lel, lol....` 

`cookie `

> COOKIE CHALLENGE 

`cookie leaderboard `

> Top Cookie users 

`cookie stat [user]`

> None 

`seed [seed]`

> See a seed 

`seed create [seed] | [title] | [description] | [image]`

> Create a seed 

`seed delete <seed>`

> Delete a seed 

`seed list `

> See all seeds 

`profile [member]`

> See a user's profile 

`profile list `

> See who has a profile in the actual guild 

`profile description <description>`

> Set profile's description 

`profile create `

> Create your profile 

`profile image [image_url]`

> Set profile's image, use `profile image remove` to remove the image 

`profile delete `

> Delete your profile 

`autotrigger <trigger>`

> See a response giving a trigger 

`autotrigger create <trigger> <response>`

> Create an auto trigger 

`autotrigger delete <trigger>`

> Delete an auto trigger 

`autotrigger list `

> See your auto triggers
